Toda TAG em HTML ou JAVASCRIPT fica entre "<" e ">"
Para escrever titulos utilizamos => "h1"
Para interpretarmos acentos utilizamos "meta charset="UTF-8""
Para fazermos link e direcionar para outra pagina utilizamos "a"
Para Fazermos o navegador interpretar JAVASCRIPT, colocamos "script" antes do codigo
Para Fazermos um pop-up utilizamos "alert" e a mensagem entre parentes e aspas "("exemplo")"
